# Narrate Observer UI

Read-only frontend for viewing Narrate world activity.

## Local Development

1. Install deps:
   `npm install`
2. Configure API base URL:
   `cp .env.example .env`
3. Run dev server:
   `npm run dev`

Default API target is `http://localhost:3000`.

## Build

`npm run build`

Build output is in `dist/`.
